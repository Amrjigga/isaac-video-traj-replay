# Isaac Video Trajectory Replay\n\nThis repo contains the first working Isaac Lab viewer for replaying hand trajectories extracted from Intel RealSense D435i bag recordings.\n\nThe current script loads a RealSense plus MediaPipe hand trajectory JSON, spawns a frozen Unitree G1 Inspire robot, and draws moving hand stick figures from the JSON landmarks.\n\nNo robot control happens yet. The robot is only spawned as a frame-alignment reference.\n\n## Main Script\n\nscripts/replay_realsense_hand_stickfigures_g1.py\n\n## Default JSON\n\nThe script expects this JSON by default:\n\n~/Downloads/20260430_133659_soft_kinematic_ready_realsense_d435i_mediapipe_hand_trajectory_v1.json\n\n## Final Working Run\n\nRun inside IsaacLab:\n\ncd ~/IsaacLab_5\n\n./isaaclab.sh -p scripts/replay_realsense_hand_stickfigures_g1.py \\n  --device cpu \\n  --axis_map forward_z_flip_x \\n  --align_yaw_deg -90 \\n  --view side_neg_y \\n  --wrist_smooth_alpha 1.0 \\n  --wrist_jump_limit 999 \\n  --hand_shape_scale 0.65 \\n  --wrist_motion_scale 1.0 \\n  --depth_motion_boost 1.0 \\n  --loop\n\n## Coordinate Fixes\n\nThe RealSense JSON uses the color optical frame:\n\n- camera x = right in image\n- camera y = down in image\n- camera z = forward from camera\n\nThe mapping that made the local hand motion and thumb side correct was:\n\nout = [x, -y, z]\n\nThis is exposed as:\n\n--axis_map forward_z_flip_x\n\nThen the whole replay trajectory needed to be rotated into the robot action-space frame:\n\n--align_yaw_deg -90\n\nFinal interpretation:\n\n- forward_z_flip_x fixes the RealSense-to-Isaac local hand coordinate mapping\n- align_yaw_deg -90 aligns the trajectory with the Unitree G1 robot frame\n\n## RealSense Data Quality\n\nThe RealSense D435i JSON was much better than the old stereo-camera output.\n\nUseful diagnostics from the successful JSON:\n\n- right hand detections: 241 frames\n- right average valid 3D landmarks: about 20.45 / 21\n- right median wrist depth: about 0.638 m\n- right median palm width: about 0.060 m\n- right wrist jump p90: about 0.017 m\n- right wrist jump max: about 0.046 m\n\n## Next Steps\n\n1.
